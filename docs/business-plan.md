## 사업계획서

---

## 1. Executive Summary

**Draft**는 대학생/청년 창업자를 위한 **AI 기반 팀빌딩 + 창업 이벤트 추천 플랫폼**입니다.

| 항목 | 내용 |
|-----|-----|
| **문제** | 좋은 아이디어가 있어도 함께할 팀원을 찾기 어려움 |
| **솔루션** | AI가 비전/스킬/성향을 분석해 최적의 팀원 매칭 |
| **타겟** | 대학생, 예비창업자, 청년 개발자/디자이너/기획자 |
| **비즈니스 모델** | 프리미엄 구독 + 기업 채용 연계 + 이벤트 주최자 광고 |
| **개발 현황** | MVP 60% 완료 (핵심 기능 구현 중) |
| **목표** | 2026년 내 MAU 10,000명 달성 |

---

## 2. 문제 정의 (Problem)

### 2.1 창업 생태계의 팀빌딩 문제

```
"아이디어는 있는데 같이 할 사람이 없어요"
"해커톤 나가고 싶은데 팀을 어디서 구하죠?"
"개발자인데 기획자/디자이너를 찾고 싶어요"
```

#### 현재 청년 창업자들이 겪는 문제:

| 문제 | 설명 |
|-----|-----|
| **팀원 탐색의 어려움** | 대학 동아리, 지인 네트워크에 의존 → 제한적인 풀 |
| **스킬 미스매치** | 막상 팀을 구해도 실력/역할 불일치로 공중분해 |
| **비전 불일치** | 단기 프로젝트 vs 진지한 창업, 목표가 달라 갈등 |
| **정보 분산** | 해커톤/창업대회 정보가 여러 플랫폼에 흩어져 있음 |

### 2.2 시장 검증 데이터

- 연간 대학생 창업 동아리: **2,000개+**
- 연간 해커톤/창업대회: **500회+**
- K-스타트업 지원자: 연 **50,000팀+**
- 팀 해체율 (창업 1년 내): **약 40%** (팀 갈등이 주요 원인)

## 3. 솔루션 (Solution)

### 3.1 핵심 서비스: Draft

**"AI가 당신의 공동창업자를 찾아드립니다"**

#### 3대 핵심 기능:

```
┌─────────────────────────────────────────────────────────┐
│                      DRAFT 플랫폼                        │
├───────────────┬───────────────┬─────────────────────────┤
│  🤝 팀빌딩     │  📅 이벤트     │  💡 아이디어            │
│  AI 매칭      │  자동 수집     │  캡처 & 발전            │
├───────────────┼───────────────┼─────────────────────────┤
│ • Opportunity │ • 해커톤      │ • Idea Snap (사진/메모) │
│   Board       │ • 창업대회    │ • AI 인사이트 생성      │
│ • AI 추천     │ • 네트워킹    │ • PRD 자동 생성         │
│ • 지원/매칭   │ • AI 추천     │                         │
└───────────────┴───────────────┴─────────────────────────┘
```

### 3.2 AI 매칭 알고리즘

```
최종 매칭 점수 =
    스킬 매칭 (40%) +
    비전 유사도 (30%) +
    실용 호환성 (20%) +
    역할 매칭 (10%)
```

| 요소 | 설명 | 기술 |
|-----|-----|-----|
| **스킬 매칭** | 내 스킬 vs 팀이 필요한 스킬 | 태그 기반 + 레벨 가중치 |
| **비전 유사도** | 창업 방향성, 관심 분야 일치도 | Gemini 임베딩 + pgvector 코사인 유사도 |
| **실용 호환성** | 위치, 시간 투자량, 온/오프라인 선호 | 조건 필터링 |
| **역할 매칭** | 원하는 포지션과 모집 역할 일치 | 1:1 매핑 |

### 3.3 차별화된 온보딩: AI Deep Dive

기존 서비스: 스킬/경력만 입력 → 표면적 매칭

**Draft**: AI 대화형 인터뷰로 **비전, 가치관, 일하는 스타일**까지 파악

```
AI: "어떤 문제를 해결하고 싶으세요?"
AI: "이상적인 팀 문화는 어떤 모습인가요?"
AI: "실패했을 때 어떻게 대처하시나요?"
    ↓
비전 요약 + 임베딩 생성 → 더 정확한 매칭
```

### 3.4 이벤트 자동 수집 시스템

| 소스 | 콘텐츠 | 상태 |
|-----|--------|------|
| 콘테스트코리아 | 공모전/창업대회 | ✅ 연동 완료 |
| 링커리어 | 대외활동/공모전 | ✅ 연동 완료 |
| 온오프믹스 | 네트워킹/세미나 | ✅ 연동 완료 |
| DevPost | 글로벌 해커톤 | ✅ 연동 완료 |
| K-Startup | 정부 지원사업 | 🔜 예정 |

**AI가 사용자 프로필 분석 → 맞춤형 이벤트 추천**

### 3.5 Idea Snap (모바일 앱)

일상에서 발견한 **문제/아이디어를 사진+메모로 즉시 캡처**

```
📸 불편한 상황 사진 촬영
📝 머릿속 해결책 메모
    ↓
AI가 파편들을 분석 → 인사이트 생성
    ↓
Draft 플랫폼에서 팀원과 함께 발전
```
