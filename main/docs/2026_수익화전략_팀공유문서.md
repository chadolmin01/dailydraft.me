## 📌 Executive Summary

**제품**: 대학·스타트업 팀의 Discord·GitHub·활동 기록을 corpus로 학습해, **3계층 페르소나(조직·프로젝트·개인)** 를 자동 구축하고 외부 SNS·모집·제안서 발행까지 자동화하는 **조직 OS**입니다.

**수익 모델**: AI 크레딧 선구매 + 구성원 할당 구조 (Free/Standard/Pro) + B2B 학교 라이선스.

**시장**: 한국 TAM 연 1조+, 글로벌 TAM 연 100조+ (Collaboration Software $102B × 학생·시드팀 버티컬).

**확장 전략**: **Land & Expand 지그재그 3단계** — 국내 대학 → 국내 창업팀 → 해외 대학/창업팀.

**2026 목표**: 매출 ₩10M + 시드 라운드 3~5억 클로징.

**최종 북극성**: Linear가 개발자 버티컬로 $1.25B 됐듯, Draft는 **비개발자 버티컬**로 동일 트랙.

## 1. 왜 이 전략인가 (검증된 문제 정의)

### 1-1. Gen Z 조직의 현실

**Deloitte 2025 Gen Z·밀레니얼 Survey**:
- 57% Gen Z / 56% 밀레니얼이 **GenAI를 일상 업무에 사용**
- **70%**가 "직장 제공 도구 수 과잉에 압도됨"
- 65%가 "개인 도구가 회사 도구보다 빠르다"

Gen Z는 **이미 Discord·Slack·개인 AI**로 일합니다. 회사가 제공한 엔터프라이즈 SaaS는 무겁고 느립니다. 이 GAP이 Draft의 진입점입니다.

### 1-2. 국내 시장 공백

최근 1년 주요 이벤트:
- **2024.1 카카오엔터프라이즈 매각** → 카카오워크 → 디케이테크인 인수
- **2024 토스랩(잔디) CB 누적** 매출 정체
- **2025.12.31 디스콰이엇 폐업** → 픽셀릭에 인수
- **Figma -81%, Monday -73%, Airtable -66%** ZIRP 거품 조정
- **Notion 한국 20명 직원 + 첫 외국어=한국어** (한국 시장 중요성 공식화)

→ **대학~중소 창업팀 세그먼트 = 실질적 무주공산**
→ Notion과는 보완 관계 포지셔닝 (정면충돌 아님)

### 1-3. "페르소나 엔진"의 왜 지금

1. **AI 페르소나 상용화** (GPT-4급 저렴해짐, 클럽당 원가 월 $0.05 검증)
2. **Gen Z 원격근무·사이드프로젝트 폭증** (팀 수 역대 최다)
3. **Discord·Slack이 팀 OS로 자리잡음** (그 위에 얹을 레이어 기회)
4. **Writer.com($47M ARR) / Jasper($35M→$180M) 엔터프라이즈 브랜드 보이스 시장 확인** → 이를 **조직(동아리·팀) 단위로 압축**한 제품은 국내외 전무

## 2. 제품 개요

### 2-1. 3계층 페르소나 엔진

```
개인 페르소나 (학생·직원)          ← 졸업·이직해도 따라감
  ↓ 속함 (맥락만 상속)
프로젝트 페르소나 (팀)             ← 조직 맥락만 상속 후 자유
  ↓ 속함
조직 페르소나 (동아리/회사)         ← 관리자 관리, corpus 최상위
```

### 2-2. 핵심 차별화

- **"mirra.my는 계정을 판다, Draft는 조직 운영권을 판다"**
- 조직 내 회장·팀장→팀원으로 크레딧이 **계층 분배** 되는 구조는 mirra·Writer·Jasper 모두 없음
- Discord corpus + GitHub DevSync 학습 → 조직 고유 목소리 자동 생성
- 승인 루프 재활용 (기존 Discord DM 버튼)

### 2-3. R1~R5 릴리즈 계획

| 릴리즈 | 내용 | 검증 기준 |
|-------|------|---------|
| R1 - Persona Core | DB 모델 + 상속 엔진 + 수동 입력 UI | FLIP 수동 작성 완료 |
| R2 - Auto Extract | Discord → 슬롯 자동 추출 | 70% 자동 채움 |
| R3 - Output Studio | 템플릿 5개 + Discord 승인 루프 | 주 2회 발행 |
| R4 - External Publish | 인스타 커넥터 | 승인 즉시 자동 포스팅 |
| R5 - 3계층 확장 | 프로젝트·개인 페르소나 오픈 | FLIP 3팀 운영 |

## 3. 🎯 지그재그 Land & Expand 전략

### 3-1. 3단계 확장 구조

```
[깊이 축] 사용자 성장 따라가기 (Vertical Depth)
[넓이 축] 지리적 복제 (Horizontal Breadth)

      1단계              2단계               3단계
    국내 대학    ──→    국내 창업생태계    ──→    해외
      (깊게)           (사용자 성장)          (복제)
        ↓                   ↓                   ↓
    같은 지역          같은 지역↑           다른 지역
    같은 사용자        같은 사용자↑         같은 사용자
```

**1단계 → 2단계**: 사용자가 커가는 걸 따라가기 (수직)
**2단계 → 3단계**: 성숙한 플레이북을 다른 지역에 복제 (수평)

### 3-2. 검증된 성공 벤치마크

| 회사 | 1단계 Land | 2단계 Expand Vertical | 3단계 Expand Geo |
|------|----------|--------------------|-----------------|
| **Toss** | 국내 P2P 송금 | 국내 금융 슈퍼앱 | 베트남·인도네시아 |
| **당근마켓** | 판교 동네 중고 | 전국 동네 생활 | 일본·캐나다·미국 |
| **Notion** | SF 개인 유저 | 미국 팀·기업 | 글로벌 |
| **Slack** | SF 게임 스타트업 | 미국 테크기업 | 글로벌 엔터프라이즈 |
| **Linear** | 인디 개발자 | 스타트업 표준 PM ($100M ARR) | (진행 중) |

**공통 패턴**: 한 지역에서 성숙·검증 뒤 지리적 복제. **해외를 먼저 안 함**.

### 3-3. 4단계 상세 로드맵

| 단계 | 기간 | 타깃 | ARR 목표 | 자금 라운드 | 팀 규모 |
|------|------|-----|---------|-----------|--------|
| **Phase 1**: 국내 대학 PMF | 2026 | 대학 동아리 + 창업동아리 | **₩30M** | 시드 3~5억 | 3~5명 |
| **Phase 2**: 국내 창업팀 전이 | 2027 | 졸업 직후 시드팀 + 액셀러레이터 팀 | **₩300~500M** | Pre-A 10~15억 | 8~12명 |
| **Phase 3**: 국내 시드~A 스타트업 | 2028 | 5~50인 스타트업 | **₩3B** | 시리즈A 30~50억 | 15~25명 |
| **Phase 4**: 해외 확장 (일본→동남아→북미) | 2029~2030 | 아시아 학생·창업팀 | **₩15B+** | 시리즈B 100~150억 | 40명+ |

### 3-4. 해외 확장의 현실 (검증된 장벽)

**직접 B2B 대학 계약 진입은 매우 어려움**:
- 미국: Modern Campus 1,400 대학 + Presence 250개 incumbent
- 영국: MSL 130+ SU 20년 장악, NUS 정치력
- Anthology는 80개국 확장 후 **2025.10 파산**
- 에브리타임 해외 진출 시도 흔적 없음

**현실적 해외 경로**:
1. **일본 서클 시장** (전용 SaaS 부재 = 공백, QANDA B2C 플레이북)
2. **동남아** (Ruangguru 모델, 채널 파트너 필요)
3. **북미** (Modern Campus·Anthology 대체자로 **인수 유도 포지션**)

**Buffer 모델 ($31M 매출, 25% 마진, 자력 성장)** 이 Draft에 더 현실적.
