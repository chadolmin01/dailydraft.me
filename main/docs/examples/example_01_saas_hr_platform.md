# PSST 사업계획서 예시 #1
## AI 기반 중소기업 채용 플랫폼 "하이어랩(HireLab)"

---

### Executive Summary

**【하이어랩】 AI 기반 중소기업 맞춤 채용 솔루션**

중소기업 인사담당자의 73%가 채용에 과도한 시간을 소요하고 있습니다. 하이어랩은 AI 매칭 알고리즘으로 서류 검토 시간을 80% 단축하고, 채용 후 6개월 내 퇴사율을 32%에서 12%로 낮춥니다.

- **시장규모**: 국내 HR테크 시장 2.3조원 (연 18% 성장)
- **현황**: MVP 완료, 베타 고객사 47개, 채용 성공률 78%
- **팀**: 삼성SDS 인사시스템 개발 출신 CTO + 사람인 PM 출신 CEO

→ 사업화 자금 1억원으로 정식 서비스 출시 및 100개 고객사 확보 목표

---

## 1. Problem (문제 인식)

### 1.1 핵심 문제 정의

> "채용에만 한 달, 뽑아놓으면 3개월 만에 퇴사... 다시 처음부터"
> — 50인 규모 IT기업 인사담당자 인터뷰 中

**중소기업 인사담당자의 73%가 '채용에 너무 많은 시간이 소요된다'고 응답했습니다.**
*(출처: 중소기업중앙회, 2025년 중소기업 인력 실태조사, N=1,200)*

### 1.2 문제의 심각성 (정량 데이터)

| 지표 | 수치 | 출처 |
|------|------|------|
| 1명 채용 평균 소요 시간 | 45일, 실무 23시간 | 사람인 HR연구소, 2025 |
| 서류 검토 비중 | 전체 시간의 40% | 자체 조사, N=150 |
| 채용 후 6개월 내 퇴사율 | 32% | 고용노동부, 2024 |
| 잘못된 채용 1건당 비용 | 2,400만원 | 한국경영자총협회, 2024 |

### 1.3 기존 솔루션의 한계

| 기존 솔루션 | 한계점 |
|-------------|--------|
| 대형 채용 플랫폼 (사람인, 잡코리아) | 중소기업 맞춤 기능 부재, 대기업 위주 UI/UX |
| 헤드헌터 | 연봉의 20-30% 수수료, 중소기업에 비용 부담 |
| 자체 채용 | 전문성 및 시간 부족, 네트워크 한계 |

### 1.4 문제 해결 시 기대 효과

- 채용 담당자 업무 시간 **80% 절감** → 핵심 업무 집중 가능
- 직무-역량 매칭 정확도 향상 → 조기 퇴사율 **20%p 감소**
- 중소기업 연간 채용 비용 **평균 40% 절감**

## 2. Solution (해결책)

### 2.1 솔루션 핵심 가치

**하이어랩은 AI 역량 매칭 알고리즘을 통해 중소기업의 채용 시간을 80% 단축하고,
직무 적합도가 높은 인재를 매칭하여 조기 퇴사율을 60% 감소시킵니다.**

### 2.2 작동 방식 (How it Works)

```
[1단계] 직무 분석        [2단계] AI 매칭         [3단계] 인터뷰 지원
    ↓                        ↓                       ↓
직무기술서 입력 →      역량 기반 후보자 →      화상면접 + AI 분석 →
AI가 핵심역량 추출     자동 스코어링/랭킹      면접 리포트 자동 생성
```

### 2.3 핵심 기능

| 기능 | 설명 | 고객 혜택 |
|------|------|----------|
| **AI 서류 스크리닝** | 이력서-직무기술서 자동 매칭 | 서류 검토 시간 90% 단축 |
| **역량 기반 매칭** | 스킬, 경험, 문화 적합도 분석 | 직무 적합 인재 채용 확률 2배 |
| **화상면접 + AI 분석** | 면접 내용 자동 요약, 평가 보조 | 면접관 간 평가 편차 감소 |
| **채용 애널리틱스** | 채용 퍼널 분석, 이탈 구간 파악 | 데이터 기반 채용 프로세스 개선 |

### 2.4 경쟁사 대비 차별화

| 비교 항목 | 하이어랩 | 대형 플랫폼 | 헤드헌터 |
|-----------|----------|-------------|----------|
| 중소기업 맞춤 | ✅ 특화 | ❌ | ❌ |
| AI 매칭 정확도 | 92% | 65% | 수동 |
| 월 비용 (50인 기준) | 29만원 | 무료~100만원 | 건당 500만원+ |
| 채용 후 관리 | ✅ 온보딩 연계 | ❌ | ❌ |

### 2.5 개발 현황 및 Traction

- **MVP 완료** (2025.09)
- **베타 테스트**: 47개 고객사, 누적 채용 진행 건수 312건
- **핵심 지표**:
  - 채용 성공률: 78% (업계 평균 45%)
  - 평균 채용 소요 시간: 12일 (기존 45일 대비 73% 단축)
  - 고객 만족도 (NPS): +62
