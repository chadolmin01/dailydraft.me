발견된 문제점

  • HIGHHIDDEN 라우트 9개 + 레거시 3개 = 죽은 코드 12개 라우트 잔존
  • HIGH고아 라우트 4개: /guide, /landing, /offline, /dev/onboarding — 진입 경로 없음

  • HIGH데스크탑 프로필 드롭다운의 "설정" 버튼이 disabled 상태 — 클릭 불가

네비게이션 비교: 데스크탑 vs 모바일

경로데스크탑 GNB모바일 상단모바일 하단 탭바문제
/explorepill tabDraft 로고중앙 탭-
/clubspill tab-없음모바일 진입 불가
/profilepill tab + 드롭다운-탭-
/projectspill tab-탭-
/projects/newCTA 버튼+ 버튼--
/notifications드롭다운벨 아이콘탭-
/messages--/more 경유직접 접근 없음
/more--탭데스크탑에서 접근 불가
검색유니버설 검색바없음-모바일 검색 불가

고아 라우트 (진입 경로 없음)

라우트상태비고
/guide공개어디서도 링크 없음
/landing공개/과 별도 존재, 어디서도 링크 없음
/offline공개PWA 오프라인 폴백 전용
/dev/onboarding개발용프로덕션 불필요
/dashboardHIDDEN내부 링크 8개 있으나 middleware 차단
/profile/edit리디렉트즉시 /profile로 리디렉트

주요 플로우

플로우경로
신규 유저/ → /login → OAuth → /auth/callback → /onboarding → /onboarding/interview → /explore
기존 유저/ → (middleware) → /explore
프로젝트 생성/projects → /projects/new → /p/{id}
클럽 진입 (데스크탑)GNB 클럽 탭 → /clubs → /clubs/{slug}
클럽 진입 (모바일)/explore 캠퍼스맵 → /clubs/{slug} (목록 페이지 우회)
Discord 연동/connect/discord → /api/discord/oauth → Discord → callback → /clubs
프로필 수정/profile → /onboarding?mode=redo-chat (온보딩 재진입)
